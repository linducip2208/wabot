Hubungkan WooCommerce ke WABot untuk notifikasi order otomatis via WhatsApp, Telegram, Instagram, dan 12+ channel.

== Description ==
Plugin ini menghubungkan toko WooCommerce Anda ke WABot — platform omnichat SaaS.

Notifikasi otomatis untuk:
- Pesanan baru
- Status order berubah (processing, completed, cancelled, dll)
- Abandoned cart recovery
- Notifikasi ke admin via WhatsApp

== Installation ==
1. Upload folder `wabot-woocommerce` ke `/wp-content/plugins/`
2. Aktifkan plugin di menu WordPress 'Plugins'
3. Buka Settings → WABot WC
4. Masukkan Webhook URL dan API Key dari dashboard WABot
5. Pilih event yang ingin dinotifikasi

== Get Webhook URL ==
1. Login ke dashboard WABot
2. Buka Integrations → E-Commerce
3. Klik Add Store → pilih WooCommerce
4. Copy Webhook URL dan API Key
